# Puppeteerの導入
## ■ Instructions
### ○ nodejs・npmの導入
下記を実行

```
$ sudo apt-get update
$ curl -sL https://deb.nodesource.com/setup_10.x | sudo -E bash -
$ sudo apt-get install -y nodejs
```

~※下記のコマンドではエラー発生した~
```
$ sudo apt-get install -y nodejs npm
$ sudo npm cache clean
$ sudo npm install npm n -g
$ sudo n stable
```
