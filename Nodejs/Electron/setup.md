# Setup - Electron-vue
## ■ Instructions
### ○ vue.jsの導入
```Shell
$ npm install -g vue-cli
$ vue init simulatedgreg/electron-vue **Project Name**
```

### ○ vue.jsの導入
```Shell
$ cd **Project Name**
$ npm install
```

### ○ アプリケーションの実行
```Shell
$ npm run dev
```

### ○ アプリケーションのビルド(./build/下に .dmg / .zip の生成)
```Shell
$ npm run build
```
